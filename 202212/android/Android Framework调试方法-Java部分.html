发表于2022-12-18|更新于2023-03-24|Android源码
|阅读量:

Java 代码的调试，主要依据两个因素，一个是你要调试的进程；一个是调试的类对应的包名路径，同时还要保证你所运行的手机环境和你要调试的代码是匹配的。

注意：符合以上两个因素，即使源码编译不过也可以调试。
